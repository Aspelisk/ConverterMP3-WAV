# ConverterMP3-WAV

# 🎵 MP3 → WAV Converter (Java)

Простая консольная утилита на Java для конвертации аудиофайлов из **MP3** в **WAV**. Программа использует `ffmpeg` для декодирования и преобразования аудио, автоматически создаёт папку вывода и сообщает о результатах работы. Подходит для быстрого пакетного преобразования файлов без потери качества.

Для работы программы необходимо установить **ffmpeg** и добавить его в системную переменную `PATH`. Скачайте сборку с [ffmpeg.org](https://ffmpeg.org/download.html): на Windows распакуйте архив и добавьте путь к папке `bin` в переменную `PATH`; на Linux выполните `sudo apt install ffmpeg`; на macOS — `brew install ffmpeg`. Проверьте установку командой `ffmpeg -version` — если отображается версия, всё настроено верно.

Скомпилируйте и запустите программу:
```bash
javac -d out Mp3ToWavConverter.java
java -cp out Mp3ToWavConverter
```
Исходный файл `hello.mp3` поместите в папку `assets/input/`, результат появится в `assets/output/result.wav`. При необходимости измените имена файлов прямо в коде (константы `INPUT_FILE_NAME` и `OUTPUT_FILE_NAME`).

> 💡 **Важно**: без установленного и доступного в `PATH` ffmpeg программа не сможет выполнить конвертацию. Если возникают ошибки — проверьте пути к файлам и права на запись в папку вывода.
